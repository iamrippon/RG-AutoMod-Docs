This document tracks all updates, improvements, and major changes made to the bot.

# 🟦 v1.0.0 — Public Release (Initial Launch)

### 🚀 Core Features
- Complete automatic link detection system  
- Real-time link deletion with customizable warnings  
- Default whitelist (essential safe domains only)  
- Custom whitelist support  
- Channel allowlist system  
- Permanent & temporary trusted user system  
- Link deletion logging system  
- Welcome message on server join  
- Dynamic status rotation (updates every 30 seconds)  

### ⚖️ Moderation Features
- Optional auto-punishment system  
  - Strike-based  
  - Timeout enforcement  
  - Automatic strike reset  
- Strike tracking with `/reset-strikes`

### 🛠 Configuration Commands
- Set/remove log channel  
- Set/reset custom warning  
- Add/remove/reset whitelist  
- Allow/block channels  
- Trust/untrust users  
- Detailed status dashboard via `/status`

### 💾 Storage System
- Full persistent JSON data storage  
- Trusted users saved (permanent + temporary)  
- Channels, whitelist, config, warnings saved automatically  
- Automatic data load on startup  

### 📚 Documentation
- Full GitHub documentation added  
- Features, commands, setup guide, permissions
