# 🤖 RG AutoMod — Discord Link Protection Bot  

RG AutoMod is a professional, fast, and highly configurable link moderation bot for Discord.  
It automatically blocks unwanted links, supports trusted users, custom whitelists, channel allowlists, logging, punishment system, and more.

This repository contains **documentation only**.  
⚠️ The bot’s source code is private and licensed.

---

## 🌟 Key Features
- 🚫 Automatic link blocking  
- 🔒 Default whitelist (17+ trusted domains)  
- 👤 Temporary & permanent trusted users  
- 📺 Channel allowlist  
- 📝 Detailed link deletion logging  
- ⚖️ Auto-punishment system (optional)  
- 💬 Custom warning messages  
- 📊 Status dashboard (/status)  
- 🔄 Dynamic activity status  
- 💾 Full persistent storage  
- 👋 Welcome message on server join
